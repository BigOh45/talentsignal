# TalentSignal

AI-powered resume triage platform for enterprise HR.

TalentSignal automates high-volume resume screening by combining structured job requirements with LLM-based candidate evaluation. It surfaces ranked shortlists with explainable scores, reducing time-to-shortlist while maintaining consistent, auditable hiring criteria.
