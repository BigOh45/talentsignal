Don't Miss Good Candidates — Protection Mechanisms
Soft Floor Only
Only Stage 1 is a hard reject. Everything else is ranked, never deleted. Thin pipeline? Recruiter dips into lower tiers.
Adjacent Skills
Prefect ≠ rejected for Airflow. Flagged as "adjacent skill" with note. Recruiter sees nuance, not a red cross.
Parse Failures → Human
Confidence < 0.6 = manual review queue, never silently dropped. Scanned CVs always get human eyes.
Weekly Audit
20 random Tier 3 CVs sampled weekly for senior recruiter spot-check. Catches any systematic gaps in SKILL.md rules.
